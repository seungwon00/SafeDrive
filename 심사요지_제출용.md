- [ ] GPS 기반 실시간 속도 측정 정확도

- [ ] 운전 습관 분석 기능
- [ ] **보호자 등록 및 관리 기능** (1차 예방)

## 4. 안전성 및 응급 대응 기능 (15점)

### ✅ 평가 항목

#### 1차 예방 기능 (사전 대비)
- [ ] **보호자 등록 기능**: Settings 화면에서 보호자 이름 및 전화번호 등록
- [ ] 보호자 정보 저장 (SharedPreferences 암호화 저장)
- [ ] 등록된 보호자 정보 확인 및 수정 기능
- [ ] 보호자 등록 상태 표시 (등록됨/미등록)

#### 2차 대응 기능 (실시간 긴급 알림)
- [ ] **급발진 감지 시 실시간 문자 알림**: 위험 상황 발생 즉시 보호자에게 SMS 자동 전송
- [ ] SMS 알림 내용: 급발진 발생 사실, 현재 속도, 위치 정보 (위도/경도, Google Maps 링크)
- [ ] 백그라운드 자동 전송 (사용자 개입 불필요)
- [ ] GPS 위치 정보 정확도 검증 후 전송
- [ ] 급발진 감지 시 경고 시스템 작동 (화면 경고, 음성 안내)
- [ ] 차량 타입별 정지 방법 안내 (일반 차량/스마트키 차량)

### 📋 상세 기능 설명

#### 보호자 등록 프로세스
1. **설정 화면 접근**: Settings 탭 → Account 섹션 → "보호자 설정" 선택
2. **정보 입력**: 보호자 이름 및 전화번호 입력 (유효성 검증)
3. **저장**: 입력한 정보는 암호화되어 SharedPreferences에 저장
4. **상태 확인**: Settings 화면에서 등록 상태 확인 가능 ("등록됨: [이름]" 또는 "미등록")

#### 긴급 SMS 알림 프로세스
1. **감지**: 급발진 감지 알고리즘이 위험 상황 감지 (신뢰도 50% 이상)
2. **자동 전송**: 보호자 정보가 등록되어 있으면 자동으로 SMS 전송
3. **알림 내용**:
   ```
   [SafeDrive 긴급 알림]
   
   [보호자 이름]님, 
   운전자에게 급발진 의심 상황이 발생했습니다!
   
   현재 속도: [속도] km/h
   위치: https://maps.google.com/?q=[위도],[경도]
   
   운전자의 안전을 확인해주세요.
   
   - SafeDrive 안전 운전 앱
   ```
4. **백그라운드 처리**: SMS 전송은 백그라운드에서 자동 처리되어 사용자 개입 불필요

### 평가 기준
- **우수 (13-15점)**: 
  - 1차 예방(보호자 등록) 및 2차 대응(실시간 SMS) 기능이 완벽하게 작동
  - SMS 전송이 정확하고 즉시 이루어짐
  - 위치 정보가 정확하게 포함됨
- **양호 (10-12점)**: 
  - 기본 응급 기능은 작동하나 일부 오류 발생 가능
  - SMS 전송은 작동하나 위치 정보가 부정확할 수 있음
- **보통 (7-9점)**: 
  - 보호자 등록 기능만 작동하거나 SMS 전송에 문제가 있음
  - 일부 응급 기능만 구현됨
- **미흡 (0-6점)**: 
  - 보호자 등록 또는 SMS 전송 기능이 작동하지 않음
  - 응급 대응 기능이 미흡함
