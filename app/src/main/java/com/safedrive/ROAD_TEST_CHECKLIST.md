# 🚗 SafeDrive 실제 도로 테스트 체크리스트

## ⚠️ 테스트 전 필수 확인 사항

### 1. 권한 설정 ✅
- [ ] 위치 권한: "항상 허용" 설정
- [ ] SMS 권한: 허용
- [ ] 배터리 최적화: SafeDrive 앱 제외 설정

### 2. 보호자 설정 ✅
- [ ] 보호자 이름 입력
- [ ] 보호자 전화번호 입력 (실제 번호)
- [ ] 차량 종류 선택 (일반/스마트키)

### 3. 휴대폰 설정 ✅
- [ ] GPS 켜기
- [ ] 볼륨 크게 (음성 안내용)
- [ ] 화면 자동 꺼짐: 5분 이상 설정
- [ ] 데이터/Wi-Fi: 켜기 (백업용)

### 4. 차량 준비 ✅
- [ ] 휴대폰 거치대 설치
- [ ] 충전 케이블 연결
- [ ] 화면이 운전자에게 잘 보이는지 확인

---

## 🎯 테스트 순서

### Step 1: 앱 시작 (주차 상태)
1. SafeDrive 앱 실행
2. Home 화면에서 [실제 모드] 클릭
3. GPS 신호 잡힐 때까지 대기 (1-2분)
4. 센서 상태 확인:
   - ✅ GPS: 사용중
   - ✅ 가속도계: 작동중

### Step 2: 주행 시작
1. 차량 시동 걸기
2. 게이지가 0 km/h인지 확인
3. 출발
4. 속도 증가하면서 게이지 확인

### Step 3: 다양한 도로 테스트
1. **골목길/주택가** (30 km/h 이하)
   - 도로 타입: "골목길" 표시되는지 확인
   
2. **일반 도로** (40-80 km/h)
   - 도로 타입: "일반도로" 표시되는지 확인
   
3. **고속도로** (80+ km/h)
   - 도로 타입: "고속도로" 표시되는지 확인

### Step 4: 주행 종료
1. 목적지 도착
2. 차량 정지
3. History 탭에서 기록 확인
4. Analytics 탭에서 점수 확인

## 🔧 추가하면 좋은 기능

### 필수 추가:
1. ✅ **주행 시작/종료 버튼** - 수동 제어
2. ✅ **권한 요청 화면** - 최초 실행 시
3. ✅ **화면 꺼짐 방지** - 주행 중 화면 유지
4. ✅ **GPS 정확도 표시** - 실시간 신호 품질
5. ✅ **주행 시간 표시** - 현재 주행 시간

### 편의 추가:
6. ⭐ **음성 안내 음량 조절** - 설정에서
7. ⭐ **데이터 내보내기** - CSV/Excel
8. ⭐ **주행 경로 저장** - 지도에 표시

## ❌ 제거하면 좋은 것

### 실제 주행 시 방해되는 요소:
1. ❌ 테스트 버튼 (발표용) - 숨기기 또는 제거
2. ❌ 디버그 로그 - 성능 향상
3. ❌ 샘플 데이터 - 실제 데이터만 표시

## 🐛 예상 문제 및 해결

### GPS 신호 약함
→ 터널/건물 사이에서 정상
→ 하늘이 트인 곳으로 이동

### 앱이 백그라운드로 가면 정지
→ 배터리 최적화 제외 설정 필요

### 화면이 꺼짐
→ 화면 자동 꺼짐 시간 늘리기
